## 한 줄로 말하면

**AI 시대의 보안**을 같은 척도로 비교할 수 있게 만든 한국어 레퍼런스 사이트입니다.

두 방향을 다룹니다.

- **AI로 보안을 강화하는 일**: Security Copilot, Agentic SOC 같은 보조 도구
- **AI 자체를 지키는 일**: LLM 보안, Prompt Injection 방어, Agentic 보안

## 왜 만들었나

실무에서 보안 솔루션을 검토할 때 자주 마주치는 문제가 있습니다.

- **Gartner나 Forrester** 리포트는 유료라 보기 어렵습니다.
- **NIST나 CISA** 표준은 무료지만 특정 제품과의 매핑을 알려주지 않습니다.
- **벤더 백서**는 당연히 자사에 유리합니다.
- **한국어 자료**는 단편적인 후기 위주입니다.
- **AI 보안** 영역은 더 심합니다. 표준(OWASP LLM/Agentic Top 10)이 빠르게 진화하고 시장이 매달 바뀝니다.

표준을 한국 현장의 솔루션에 매핑하는 **중립 레퍼런스가 비어 있다**고 판단해 만들었습니다.

## 누가 운영하나

한국에서 일하는 익명의 보안 실무자가 개인 자격으로 운영합니다.

실명과 소속은 공개하지 않습니다. 광고나 스폰서, 제휴 링크도 받지 않습니다.

## 운영 원칙

**공개 자료만 사용합니다.** 모든 평가는 벤더 공식 문서, 표준 문서, 공개 리포트, 분석가 리포트를 근거로 합니다. 각 글의 마지막에 원문 링크를 명시합니다.

**같은 척도로 평가합니다.** [평가 척도](/evaluation/scoring-rubric/) 페이지에서 척도를 미리 공개합니다. 모든 솔루션 리뷰가 같은 척도를 따릅니다.

**벤더 중립을 유지합니다.** 광고 없음, 스폰서 없음, 제휴 링크 없음. 특정 벤더에 유리하게 쓰지 않습니다.

**신선도를 책임집니다.** AI 영역은 분기가 아니라 **월 단위**로 시장이 바뀝니다. 각 페이지 하단에 마지막 검토일을 표시하고 정기 재검토합니다.

**인용 가능하게 만듭니다.** 평가 항목마다 고유 ID(예: `ZTA-ID-AUTH-03`)를 부여해서 외부에서 정확히 인용하고 검색할 수 있게 합니다.
