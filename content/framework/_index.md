## 국제 표준

영문 표준 문서를 한국어로 요약했습니다. 원문 인용 시 한 번 더 읽기 위한 빠른 참조용입니다.

### 작성 완료

- [CISA Zero Trust Maturity Model v2](cisa-ztmm-v2/) — 본 사이트 백본
- [NIST SP 800-207](nist-800-207/) — Zero Trust Architecture 원전

### 작성 예정

- DoD Zero Trust Reference Architecture
- NIST CSF 2.0
- MITRE D3FEND
- CSA Cloud Controls Matrix
- 세 표준 비교표 (CISA, NIST, DoD)

## 한국 규제와 환경

국제 표준만 봐서는 한국 시장에 그대로 적용하기 어렵습니다. 망분리, 공공 인증, 금융 규제 환경과 함께 봐야 의미가 있습니다.

### 작성 예정

- 망분리 정책과 Zero Trust 적용 시 충돌과 해소 방안
- 공공 부문 보안 인증 체계(CSAP)와 Zero Trust 정합성
- 금감원 전자금융감독규정과 Zero Trust
- 개인정보보호법 개정과 Zero Trust Data 영역

## 본 사이트는 이 표준들을 어떻게 사용하나

| 표준 | 어디에 사용 |
|---|---|
| CISA ZTMM v2 | [솔루션 가이드](/pillars/)의 5 영역 구분, 평가 ID 체계 |

| 한국 규제 | 솔루션 리뷰의 "한국 시장 노트" 섹션 |
