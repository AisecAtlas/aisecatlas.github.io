본 사이트의 모든 솔루션 리뷰는 동일한 평가 척도를 따릅니다.

- [평가 척도(Rubric) 상세](scoring-rubric/) — A·B 두 축의 항목 정의 + ID 체계
- 🚧 방법론 — 점수 산정 절차, 출처 가중치, 검토 주기

## 핵심 원칙

1. **출처 기반**: 공개 자료(벤더 문서·표준·공개 리포트)로만 평가
2. **표준 정합성 우선**: CISA ZTMM v2 + NIST 7원칙 기준
3. **인용 가능성**: 항목마다 고유 ID
4. **신선도**: 분기별 재검토
