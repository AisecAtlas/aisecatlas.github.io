본 사이트의 모든 솔루션 리뷰는 같은 척도를 사용합니다. 어떤 항목을 어떻게 보는지 미리 공개합니다.

- [평가 척도 상세](scoring-rubric/) — 항목 정의와 ID 체계

## 핵심 원칙

**공개 자료만 봅니다.** 벤더 공식 문서, 표준 문서, 공개 리포트만 근거로 사용합니다.

**표준 정합성을 우선합니다.** CISA Zero Trust Maturity Model v2와 NIST 7원칙을 기준으로 합니다.

**고유 ID를 부여합니다.** 항목마다 ID가 있어서 외부 인용과 검색이 가능합니다.

**주기적으로 다시 봅니다.** 분기마다 재검토합니다.
