## 이 사이트가 답하는 질문

조직에서 보안을 책임지고 있다면, 2026년에 두 가지 질문이 동시에 찾아옵니다.

- **AI를 도입해 SOC 운영을 어떻게 가속할까?**
- **AI 자체가 만들어내는 새로운 공격면을 어떻게 막을까?**

AISec Atlas는 두 질문 모두에 한국어로 답합니다. CISA와 NIST의 Zero Trust 표준, OWASP LLM Top 10과 Agentic Top 10 같은 AI 특화 위협 모델을 같은 척도로 다룹니다.

> **공개 자료만 사용합니다.** 광고, 스폰서, 제휴 링크 없음. 도입을 검토하는 실무자가 의사결정에 바로 쓸 수 있도록 정리합니다.

## 4개 섹션

| 섹션 | 어떤 도움이 되나 |
|---|---|
| [솔루션 가이드](/pillars/) | CISA ZTMM 5개 영역별 카테고리와 솔루션 리뷰. Identity, Devices, Networks, Applications, Data. |
| [표준과 규제](/framework/) | CISA, NIST 표준 한국어 요약과 한국 규제 환경. 망분리, 금감원, CSAP. |
| [AI 동향](/ai/) | AI를 지키는 일(LLM 보안)과 AI로 보안 강화(Security Copilot, Agentic SOC). |
| [자동화 (with AI)](/automation/) | SOAR에서 Agentic SOC로 진화. Microsoft Security Copilot, Charlotte AI, Torq, Tines 비교. |

## 처음 방문하셨다면

조직의 관심사에 따라 다른 출발점을 추천합니다.

**일반 Zero Trust 솔루션을 검토 중이라면** → [솔루션 가이드](/pillars/) 매트릭스부터

**LLM 도입 보안을 검토 중이라면** → [AI를 지키는 일](/ai/defend-ai/)부터

**SOC 자동화를 검토 중이라면** → [Security Copilots 비교](/automation/security-copilots/)부터

**표준 문서를 빠르게 훑고 싶다면** → [CISA ZTMM v2 요약](/framework/cisa-ztmm-v2/)부터

## 평가 방식

모든 솔루션 리뷰는 같은 척도를 사용합니다. **두 축으로 10점씩, 종합은 평균**입니다.

평가 항목마다 `ZTA-{영역}-{기능}-{번호}` 형식의 고유 ID가 있어서 외부에서 정확히 인용할 수 있습니다. 척도 정의는 [평가 척도 페이지](/evaluation/scoring-rubric/)에서 미리 공개합니다.
