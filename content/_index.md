## 이 사이트가 답하는 질문

조직에 새 보안 솔루션 도입을 검토 중이라면 이런 질문을 받게 됩니다.

- 이 제품이 정말 Zero Trust 기준에 맞나요?
- 비슷한 제품 중 어떤 게 우리 환경에 맞나요?
- 영문 백서와 표준을 다시 읽어야 하나요?

AISec Atlas는 이런 질문에 한국어로, 공개 자료만 가지고, 같은 척도로 답합니다.

## 무엇을 제공하나

| 섹션 | 어떤 도움이 되나 |
|---|---|
| [매트릭스](/matrix/) | 5개 보안 영역과 4단계 성숙도를 한 화면에서 본다 |
| [영역별 가이드](/pillars/) | Identity, Devices 같은 영역 단위로 솔루션 카테고리를 확인한다 |
| [솔루션 리뷰](/solutions/) | 제품별 표준 정합성 점수와 도입 판단 기준을 본다 |
| [Top 리스트](/top/) | 카테고리별 상위 5개 제품을 비교한다 |
| [표준 요약](/framework/) | CISA, NIST의 Zero Trust 표준을 한국어로 빠르게 읽는다 |
| [한국 노트](/korea/) | 망분리, 공공, 금융 규제 환경에서의 적용 포인트를 본다 |
| [평가 방법](/evaluation/) | 사이트가 사용하는 평가 척도와 점수 산정 방식을 확인한다 |

## 빠른 시작

처음 방문하셨다면 다음 순서를 권장합니다.

1. [매트릭스](/matrix/)에서 전체 구조를 한 번 본다
2. [평가 방법](/evaluation/scoring-rubric/)에서 점수가 어떻게 매겨지는지 본다
3. 관심 있는 [솔루션 리뷰](/solutions/) 하나를 읽는다
