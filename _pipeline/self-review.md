# AISec Atlas 파이프라인 자가검토 체크리스트 (loop-until-dry)

> 대상: `_pipeline\drafts\**\index.md`, `_pipeline\poc\<slug>\*`, `ledger.csv`
> 운영: aisec-reviewer 가 초안 완성 직후 사이클 1 실행, 이후 신규 결함이 나오지 않을 때까지 반복
> **종료 조건: 2회 연속 사이클에서 신규 결함 0건** → PR 생성으로 진행
> 한 항목이라도 ❌면 그 자리에서 수정 후 사이클을 다시 돈다.

---

## A. 사이트 규약 부합성 (Hugo)
- [ ] A-1. 모든 초안에 front matter 존재: `title` `description` `date` `lastmod` `tags` `ShowToc`
- [ ] A-2. `date`/`lastmod` 형식 = `YYYY-MM-DD`, 미래일자 아님(buildFuture=false)
- [ ] A-3. 파일 경로가 매핑 규칙과 일치 (solutions/weekly/ai/automation/top)
- [ ] A-4. 리프 번들은 `index.md`, 섹션 인덱스는 `_index.md` 규칙 준수
- [ ] A-5. `<div class="card-grid">` 등 raw HTML 블록의 태그가 정상 닫힘
- [ ] A-6. 내부 링크가 상대경로(`/solutions/...`) + 실제 존재 경로

## B. 출처 정확성 (1차출처 우선)
- [ ] B-1. 모든 사실 주장에 출처 URL 명시(빈 인용 0개)
- [ ] B-2. 핵심 수치·제품명·발표일은 **1차출처(공식 발표/문서/표준)** 로 뒷받침
- [ ] B-3. 2차 자료(뉴스·블로그)만 근거인 핵심 주장 0개 — 발견 시 1차로 대체 또는 "보도 기준" 명시
- [ ] B-4. 인용 URL 무작위 2개 WebFetch 생존 확인(404·리다이렉트 깨짐 없음)
- [ ] B-5. ledger 의 `url` 과 본문 인용 출처가 정합

## C. 중복·분류 (dedup / MECE)
- [ ] C-1. 신규 항목이 `ledger.csv` 기존 항목과 url 정규화 기준 중복 아님
- [ ] C-2. 기존 `content\` 페이지와 주제 중복 시 — 새 페이지가 아니라 기존 페이지 갱신으로 처리
- [ ] C-3. augment/defend 축 분류가 내용과 일치
- [ ] C-4. 한 항목은 한 콘텐츠 목적지에만 1차 배치(교차링크는 허용)

## D. 문체·정량성 (사이트 톤)
- [ ] D-1. 페이지 상단에 `> **이 페이지의 결론**:` 결론-먼저 블록 존재
- [ ] D-2. 모호어("적절히","많은","곧") 대신 수치·날짜·고유명으로 서술
- [ ] D-3. 표 우선: 비교·매핑은 산문이 아니라 표로
- [ ] D-4. 과장·마케팅 카피("최고","혁신") 배제, 중립 기술 서술

## E. POC 설계 산출물(생성된 경우)
- [ ] E-1. 5종 세트 존재: `guide.md` `eval.csv` `eval.xlsx` `item_details.md` `selfreview_checklist.md` `summary.md`
- [ ] E-2. `eval.csv` 스키마 정합(대분류,ID,평가항목,L0~L3기준,가중치,증빙요청자료,점수,근거표준)
- [ ] E-3. Level 0~3 기준 문구가 **객관 검증 가능한 동작/구성**으로 서술(주관어 0)
- [ ] E-4. `점수` 컬럼은 공란(사람이 시연받고 채점 — H2)
- [ ] E-5. `build_eval_xlsx.py <slug>` 로 xlsx 정상 빌드(한글·자동필터·점수시트 동작)
- [ ] E-6. 모든 평가항목에 `근거표준` 채워짐(빈셀 0)

## F. 안전 게이트 (발행 경계)
- [ ] F-1. 작업 브랜치가 `intel/<주차>` 이며 `main` 아님
- [ ] F-2. 커밋·푸시 대상에 비밀/토큰/개인정보 미포함
- [ ] F-3. `main` 직접 push 시도 없음 — PR 생성으로만 종료(H1 사람 merge)
- [ ] F-4. PR 본문에 변경 요약 + 이 체크리스트 결과 + 사람 검토 포인트(H3 정확성 사인오프) 포함
